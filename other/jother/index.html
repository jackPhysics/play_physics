NUCLEAR WINDOW:

The following programs are based on the same program - Nuclear Window - which enables you to see an individual nucleus, fire particles at it and watch it decay if unstable
It has a database of all stable nuclei so it always gets stability correct and it uses an algorithm to determine decay mode for unstable nuclei that is pretty close to reality - U238 decay sequence is spot on - but Fission is not programmed in...

You start off with a Hydrogen nucleus but you input what values of protons and neutrons you want to start with, or you can set the program to continously fire in the same particle (eg protons, to simulate the situation at the heart of a star..., or alphas etc) and leave it running - will it make everything?

Nuclear Construction Kit

The Physics in this one is a bit dodgy...
Basically it is a space into which you can drag Protons (red) and Neutrons (blue) and try to put them together to make stable nuclei...
The nuclei inside the space can be rotated around so as to view it from different angles...
Unfortunately (to rather annoying sound effects) the nucleons will keep jiggling about and keep decaying into the other type of nucleon...
You can alter how the nucleons 'jiggle', whether they fall towards a common centre, what is the effective range of the nuclear and EM forces, the chances of decay if a nucleon finds itself in an 'unstable' position, etc...(warning: the "EM turn-off" button is a dummy button, it does nothing - could this program be any more annoying!?)
You can retrieve information about how many other nucleons each nucleon is touching - is this the secret of nuclear stability?.
Does any of it lead you to any deeper understanding as to why some nuclei are stable while others are unstable? Probably not.
One other interesting thing to do is to press the big red STOP button to stop all movement and then use the program as a means of investigating the packing of spheres

To further help your investigations here is a spreadsheet that lists all the stable nuclei spreadsheet
